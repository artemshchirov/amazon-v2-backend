# Backend X Amazon 2.0 X NestJS

## Description

E-commerce app where buyers can see a catalog of products or services and electronically purchase them.

- NestJS
- TypeScript
- PostgreSQL
- Prisma ORM

## Running the app

```bash
# installation
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
